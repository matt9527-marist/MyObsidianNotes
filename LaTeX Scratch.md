$$\text{Relative Displacement Vector}$$
$$D(t_{i}) = T(t_{i}) - C(t_{i})$$
$$\text{Distance to target } r_{i} = ||D(t_{i})||$$
$$\text{Relative Speed } v_{i} = ||D(t_{i}) - D(t_{i-1})||$$

t_{a} \implies \text{on target} \\
