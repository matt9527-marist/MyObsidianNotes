We had discussed *dangling references* previously. To avoid dangling references, we can use dynamic memory allocation. 

```c++
#include <iostream>
#include <string>
std::string& getName() {
	std::string *name = new std::string{"Alice"};
	return *name;
}
int main() {
	std::string& name{getName()};
	std::cout << "Hello " << name << "\n";
}
//Hello Alice
```
